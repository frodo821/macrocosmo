## 現在のアーキテクチャ要点

### データモデル
- **Amt** (u64 fixed-point ×1000) + **SignedAmt** — 全資源値
- **ModifiedValue** + **Modifier** — base_add/multiplier/add, ScopedModifiers + CachedValue
- **ResourceStockpile** は **StarSystem** Entity 上（Colony ではない）
- **Planet vs System Buildings** — Mine/Farm/PowerPlant は Colony、Shipyard/Port/ResearchLab は StarSystem
- **Ship Design** — Hull + Module + ShipDesign（Lua定義）、ShipDesignPreset はフォールバック
- **TechId(String)** — 数値から文字列に移行済み
- **Condition** — All/Any/OneOf/Not の木構造、prerequisites 評価用
- **DeepSpaceStructure** — capability ベース、Lua define_structure で定義

### 通信・情報システム
- **KnowledgeStore** — 光速遅延つき星系情報。資源スナップショット含む
- **Empire資源集計** — 遠方は KnowledgeStore（遅延）、ローカルはリアルタイム
- **PendingShipCommand** — 遠方への命令は光速遅延
- **Survey carry-back** — FTL船はデータ持ち帰り（光速より速い場合）、遅い場合は光速伝播

### 移動・戦闘
- **統一MoveTo** — FTL/SubLight自動選択、plan_ftl_route + hybrid routing
- **FTLは探索済み星系のみ** — plan_ftl_route が unsurveyed を拒否
- **ROE** — Aggressive/Defensive/Retreat、光速遅延で変更
- **プレイヤー移動** — Board/Disembark、旗艦撃沈時は首都リスポーン
- **敵対勢力** — 銀河生成時12%配置、首都近傍除外、距離スケーリング

### UI
- **全画面星系ビュー** — egui Canvas で星系マップ描画、惑星クリック選択
- **独立惑星ウィンドウ** — Order::Foreground で前面表示
- **Ship Designer** — Hull選択→Module配置→プレビュー→保存
- **Refit** — ドック中のモジュール入替（コスト・時間あり）
- **ツールチップ** — マップ + アウトライン両方でホバー表示
- **テリトリーシェーダー** — 1/r² authority field、WGSL Material2d

### Lua スクリプティング
- **単一エントリポイント** — `scripts/init.lua` → require() で依存順に読み込み
- **戻り値参照** — define_xxx が参照テーブルを返す、文字列IDも後方互換
- **サンドボックス** — io/os/debug/ffi 無効、loadfile/dofile nil
- **パス解決** — exe隣 → CWD → CARGO_MANIFEST_DIR の探索

## 設計メモ（未実装）

### Condition スコープ付き ctx (#130 残り)
```lua
prerequisites = function(ctx)
    return all(
        ctx.empire:has_tech("ftl_interdiction"),
        ctx.system:has_building("shipyard"),
    )
end
```
- Lua 関数が Condition データ構造を返す（bool ではない）
- 評価サイトが ctx を構築してスコープを解決
- 相手ありの場合も `ctx.target.empire:has_flag(...)` で対応可能

### ScopedFlags
- `GameFlags` (グローバル HashSet) → Entity に付く `ScopedFlags` コンポーネントに移行
- スコープ: Empire / System / Planet / Ship
- Condition の `has_flag` はスコープチェーンで探索

### Interdictor 設計
- IFF なし — 味方も影響
- FTL 航行 + FTL 通信 両方妨害
- オフ命令は光速のみ（FTL通信が止まってるため）
- タイマー設定 or 現地で直接操作
